Navegador web

Un navegador web (en inglés) es un sóftware, aplicación o programa que permite el acceso a la Web, interpretando la información de distintos tipos de archivos y sitios web para que estos puedan ser vistos.

La funcionalidad básica de un navegador web es permitir la visualización de documentos de texto, posiblemente con recursos multimedia incrustados.

Ejemplos de navegador web

Safari: Safari es un navegador web de código cerrado desarollado por Apple Inc. Está disponible para macOS, iPadOS e iOS.
Motzilla Firefox es un navegador web libre y de código abierto desarollado por Linux, Android, iOS, macOS y Microsoft Windows coordinado por la Corporación Mozilla y la Fundación Mozilla.
Google Chrome Google Chrome es un navegador web de código cerrado desarrolado por Google, aunque derivado de proyectos de cófigo abierto.
